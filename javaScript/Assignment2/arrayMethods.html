5 Array methods


1-> push(): this method push element into array

For example:

let fruits = ["Apple","Mango","Banana"]
//push grapes into the array
fruits.push("Grapes");


2-> pop(): this method removed element from array

For example:

let fruits = ["Apple","Mango","Banana"]
//pop Banana into the array
fruits.pop("Banana");


3-> splice(): This method can be used to add new items to an array

For example:

let fruits = ["Apple","Mango","Banana"]
//add lemon and kiwi at grapes into the array
fruits.splice(2, 0, "Lemon", "Kiwi");


4-> concat(): This method can be used to Merged to arrays

For example:

let fruits = ["Apple","Mango","Banana"]
let vegetables = ["tomato","potato","ladyfinger"]
let eatableItems = fruits.concat(vegetables);


5-> join(): This method is same as toString(), it joins all array element as a string but you can add separator.

For example:

let fruits = ["Apple","Mango","Banana"]
document.getElementById("demo").innerHTML = fruits.join(" * ");
