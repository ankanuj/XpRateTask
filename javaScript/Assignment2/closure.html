Counter Using closure.


Example of counter using Closure

suppose we have a function called displayName()
and now we have to count how many time function called using closure.
function displayName(){
var counter = 0;
function count(){
counter += 1;
document.getElementById("output").innerHTML = "No. of time function called is "+counter+" "+"times";
}
return count;
};
var display = displayName();
display();
display();
